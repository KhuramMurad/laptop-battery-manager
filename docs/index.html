Why Choose Battery Threshold Manager?

🔍

Dynamic Compatibility Check

Scans hardware controllers on launch. Adapts dynamically to battery names (`BAT0`, `BAT1`) and reports full diagnostics if unsupported.

🛡️

Secure Root Elevation

Utilizes native Polkit (`pkexec`) on Linux and Windows UAC prompts. Elevated permissions are requested only for write tasks; core UI runs securely without root.

💻

True Cross-Platform

Decoupled backends for Linux (sysfs kernel thresholds) and Windows (WMI and PowerShell interface for Lenovo, ASUS, and Dell laptops).

Simple Installation

Install the pre-compiled `.deb` package using apt to automatically configure Polkit privileges and desktop shortcuts:

# 1. Download the package
wget https://github.com/KhuramMurad/laptop-battery-manager/raw/main/docs/dist/battery-threshold-manager_1.0.0_all.deb

# 2. Install it with apt
sudo apt update
sudo apt install ./battery-threshold-manager_1.0.0_all.deb

Install the pre-compiled `.rpm` package using dnf to configure system launchers and Polkit settings:

# 1. Download the package
wget https://github.com/KhuramMurad/laptop-battery-manager/raw/main/docs/dist/battery-threshold-manager-1.0.0-1.noarch.rpm

# 2. Install it with dnf
sudo dnf localinstall battery-threshold-manager-1.0.0-1.noarch.rpm

Install the application natively using the setup installer wizard. The app will launch with a UAC elevation prompt to access WMI settings:

  1. Download the battery-threshold-manager-setup.exe installer.
  2. Double-click to launch the setup wizard and follow the steps.
  3. Launch the app via the Start Menu shortcut. UAC permissions will be requested to access battery controller interfaces.
